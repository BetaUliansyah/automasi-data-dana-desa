# automasi-data-dana-desa
Skrip untuk mendapatkan semua data dana desa dari web jaga.id.

# Work in Progress
Skrip ini masih dalam tahap pengembangan. Ada 3 file dengan kegunaan sbb:
1. daftar-desa.py untuk mendapatkan seluruh kode desa yang tersedia di jaga.id (sekitar 74 ribu desa dari 94 ribu seluruh desa se-Indonesia)
2. single-post.py untuk mendapatkan data 1 desa saja
3. loop-all-desa.py untuk mendapatkan data seluruh desa se-Indonesia (masih dalam tahap pengembangan)
